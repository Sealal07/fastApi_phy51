# Space Control API & Dashboard

Система управления космическими кораблями с полноценной авторизацией. Проект позволяет регистрировать пользователей, входить в систему и управлять флотом межзвездных кораблей через веб-интерфейс.

---

## Особенности проекта
* **Безопасность:** Хеширование паролей с помощью `bcrypt` и аутентификация через JWT-токены.
* **Управление данными:** Хранение пользователей и кораблей в реляционной базе данных PostgreSQL.
* **Автоматизация:** При регистрации корабля API автоматически рассчитывает время прибытия до Марса на основе введенной скорости.
* **Защита маршрутов:** Просмотр списка кораблей доступен всем, но добавление новых данных — только авторизованным пользователям.

---

##  Технологический стек

### **Backend (Python)**
* **FastAPI** — основной фреймворк.
* **SQLAlchemy** — ORM для работы с базой данных.
* **JOSE (JWT)** — генерация и проверка токенов доступа.
* **Passlib** — безопасное хеширование паролей.

### **Frontend (JavaScript)**
* **React** — библиотека для создания пользовательского интерфейса.
* **Axios** — выполнение HTTP-запросов к API.

---

##  Структура файлов
* `auth.py`: Настройка JWT, стратегии хеширования и функции проверки токена.
* `database.py`: Конфигурация SQLAlchemy, описание моделей `UserModel` и `SpaceshipModel`.
* `space_api.py`: Определение эндпоинтов (регистрация, логин, работа с кораблями).
* `App.jsx`: Логика фронтенда: формы входа, регистрации и отображение списка кораблей.

## Инструкция по запуску

### 1. Подготовка базы данных
Убедитесь, что у вас установлен **PostgreSQL**.
1.  Создайте базу данных `space_ship`.
2.  В файле `database.py` обновите строку подключения (укажите свой пароль):
    ```python
    DATABASE_URL = 'postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/space_ship'
    ```

### 2. Запуск Backend (API)
1.  Установите необходимые библиотеки:
    ```bash
    pip install fastapi uvicorn sqlalchemy psycopg2-binary "python-jose[cryptography]" "passlib[bcrypt]" python-multipart
    ```
2.  Запустите сервер из корневой директории:
    ```bash
    uvicorn space_api:app --reload
    ```
    *API будет доступно по адресу: `http://127.0.0.1:8000`*

### 3. Запуск Frontend (React)
1.  Перейдите в папку проекта: `cd space-front`.
2.  Установите зависимости: `npm install axios`.
3.  Запустите приложение: `npm run dev` или `npm start`.

## Основные эндпоинты API
| Метод | Эндпоинт | Описание | Доступ |
| :--- | :--- | :--- | :--- |
| `POST` | `/register_user` | Регистрация нового аккаунта | Публичный |
| `POST` | `/token` | Получение токена (Login) | Публичный |
| `GET` | `/ships` | Список всех кораблей | Публичный |
| `POST` | `/ships/register` | Добавление нового корабля | **Только с JWT** |
